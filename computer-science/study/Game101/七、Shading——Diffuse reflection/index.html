# 着色（Shading）

对不同物体，应用不同材质（material），这里暂时不考虑物体的阴影。

# 冯氏光照（Blinn-Phong）

  • 高光（Specular highlights)

  • 漫反射（Diffuse reflection）

  • 环境光照（Ambient lighting）

# 漫反射（Diffuse reflection）

# Shading point

  • 视野方向（View direction） v⃗\vec vv
  • 物体表面法线（Surface normal）n⃗\vec nn
  • 光照方向（Light direction）l⃗\vec ll
  • 物体表面参数（Surface parameters）
    • 颜色
    • 亮度
    • ...

# 角度和光线强度的关系

不同角度的光线，明暗不一样。物体表面的光线和物体表面的法线夹角决定亮度 cos⁡θ=n⋅l\cos \theta = n \cdot lcosθ=n⋅l。

# 距离和光线强度的关系

假设能量在传播过程中不会减少，假设半径为 1 的球表面上某个点的能量为 III，那么通过计算，得到半径为rrr 的球上的能量就是 Ir2\frac{I}{r^2}r2I​。

# 光线强度的最终结果

  • kdk_dkd​：表示材质表面的漫反射系数。
  • Ir2\frac{I}{r^2}r2I​：表示距离对光照强度的影响。
  • max(0,n⋅l)max(0,n \cdot l)max(0,n⋅l)：表示光线本身和法线的夹角，影响光照的强度。（不考虑 cos⁡θ\cos \thetacosθ 为负数的情况）
  • 漫反射不需要考虑观察者的视角 v⃗\vec vv。
